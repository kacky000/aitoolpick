# Turso Review 2026: Is It Worth It?

Turso is edge database built on libsql (sqlite fork) that replicates data globally for ultra-low latency reads, designed for edge computing and serverless applications. In this review, we cover everything you need to know before subscribing.

## What is Turso?

Turso is edge database built on libsql (sqlite fork) that replicates data globally for ultra-low latency reads, designed for edge computing and serverless applications. It is designed for edge applications, serverless developers, jamstack apps and anyone looking for a reliable solution in the database space.
