GitHub Copilot is the most widely used AI coding assistant, and its pricing is about to change significantly. Starting June 1, 2026, Copilot moves from request-based billing to a usage-based credit system. Here's what you need to know.

## Current Plans (Before June 2026)

| Plan | Price | Included |
|------|-------|---------|
| **Free** | $0 | 2,000 completions + 50 chat requests/month |
| **Pro** | $10/mo | Unlimited completions + premium requests |
| **Pro+** | $39/mo | Everything in Pro + more premium requests |
| **Business** | $19/user/mo | Team management + policy controls |
| **Enterprise** | $39/user/mo | Everything + IP indemnity |

## What Changes on June 1, 2026

GitHub is replacing "premium requests" with **AI Credits**. The plan prices stay the same, but what you get is measured differently.

### How AI Credits Work

- **1 AI credit = $0.01 USD**
- Each plan includes monthly credits equal to its price: Pro gets $10 in credits, Business gets $19 per user
- Credits are consumed based on the AI model used and tokens processed
- **Code completions remain unlimited** on all paid plans — they don't use credits
- **Next Edit Suggestions remain unlimited** on paid plans

### What Uses Credits

- Chat messages (in editor, on GitHub.com, in CLI)
- Copilot Edits (multi-file editing)
- Agent mode sessions
- Using premium models (Claude Sonnet, GPT-4o, etc.)

### What's Still Free/Unlimited

- Code completions (Tab suggestions)
- Next Edit Suggestions
- Using the default model for completions

## Plan-by-Plan Breakdown

### Free ($0/month)

GitHub Copilot Free gives everyone a taste:
- 2,000 code completions per month
- 50 chat requests per month (including Copilot Edits)
- Limited model selection

Good enough for students or light hobby use. Not viable for full-time development.

### Pro ($10/month)

The standard individual plan:
- Unlimited code completions
- $10 in monthly AI credits
- Access to all available models
- Copilot Chat, Edits, and Agent mode

At $10/month, this is the cheapest paid AI coding assistant on the market — half the price of [Cursor Pro](/blog/cursor-pricing-2026).

### Pro+ ($39/month)

For developers who need heavy chat and Agent usage:
- Everything in Pro
- $39 in monthly AI credits (3.9x Pro)
- Higher rate limits

### Business ($19/user/month)

Built for organizations:
- Everything in Pro for each seat
- $19 in monthly AI credits per user
- Organization-wide policy management
- Audit logs
- IP indemnity
- Exclude specified files from Copilot

### Enterprise ($39/user/month)

Everything in Business, plus:
- Fine-tuned models on your codebase
- Extended AI credits per user
- Priority support

## Migration Timeline

- **Monthly Pro/Pro+ subscribers**: Automatically move to usage-based billing on June 1, 2026
- **Annual Pro/Pro+ subscribers**: Stay on request-based billing until their current term expires
- **Business/Enterprise**: Transition managed by GitHub account teams

## Is GitHub Copilot Still Worth It?

At $10/month, Copilot Pro remains the best value in AI coding tools. The credit system may feel limiting for heavy Agent users, but unlimited completions — which is what most developers use most — stay free on all paid plans.

The real question is whether $10 of credits is enough for your chat and editing needs. If you routinely use Agent mode and premium models, the $39 Pro+ plan or a competitor like [Cursor](/blog/cursor-review-2026) may be more cost-effective.
