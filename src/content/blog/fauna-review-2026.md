# Fauna Review 2026: Is It Worth It?

Fauna is distributed serverless database offering acid transactions, document-relational model, and native graphql support designed for modern cloud and edge applications. In this review, we cover everything you need to know before subscribing.

## What is Fauna?

Fauna is distributed serverless database offering acid transactions, document-relational model, and native graphql support designed for modern cloud and edge applications. It is designed for serverless developers, jamstack apps, globally distributed apps and anyone looking for a reliable solution in the database space.

## Pros

- Globally distributed with strong consistency
- Combines document and relational models
- Native GraphQL and FQL query support
- Serverless with no infrastructure to manage

## Cons

- Proprietary query language (FQL) has learning curve
- Can be expensive at high read/write volumes
- Smaller community compared to MongoDB or PostgreSQL
