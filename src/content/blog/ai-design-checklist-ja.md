AIでサイトを作ると、どれも同じ見た目になる。青いグラデーションの背景、Interフォント、3カラムのカードが3回繰り返されて、絵文字がアイコン代わりに並んでいる。

見た瞬間に「AIで作ったな」とわかる。

この1年でいくつものサイトをAIで構築した実体験から、「AI臭い」デザインのパターンを30項目にまとめた。テストツールでは検出できない。人間の感覚でしか気づけないものばかりだ。

## なぜAIっぽいとマズいのか

GoogleのHelpful Content Updateはテキストだけを見ているわけじゃない。ページ全体のユーザー体験を評価する。テンプレそのままのデザインは「手抜き」のシグナルになる。コンテンツがどれだけ良くても。

もっと直接的に言えば：ユーザーは「よくあるサイトだな」と感じたら離脱する。AIで作った他のサイトと同じ見た目なら、デザインで差をつけるチャンスを捨てている。

## 6カテゴリの「AIっぽさ」パターン

### 1. 配色

一番バレやすいのが紫→青のグラデーション背景。AIは何千ものSaaSランディングページを学習しているから、「プロっぽい」＝このグラデーションになる。

| AIがやること | 代わりにやること |
|---|---|
| 紫→青グラデーション背景 | 単色ベース＋アクセント1色 |
| グラデーションボタン | フラットカラー＋hoverで変化 |
| 全要素にbox-shadow | 影は控えめに。borderで区切る |
| セクション背景色の交互切り替え | 本当に区切りたい場所だけ |
| 5色以上 | 白＋黒＋アクセント1色で十分 |

### 2. フォント

AIが最初に選ぶフォントは3つ: Inter、Roboto、Open Sans。どれもいいフォントだけど、使った時点で「フォントのことは何も考えてない」と宣言しているのと同じ。

| AIがやること | 代わりにやること |
|---|---|
| Inter/Roboto/Open Sans | 日本語: Noto Sans JP, BIZ UDGothic。英語: DM Sans, Sora等 |
| 見出しも本文も同じフォント | ファミリーを変えるか、spacing・weightで差をつける |
| 文字サイズが均等刻み（16→24→32） | 見出しをドカンと大きく。メリハリ重視 |
| 全てline-height: 1.5 | 見出し: 1.1〜1.3、本文: 1.6〜1.8 |

### 3. レイアウト

3カラムの同じカードが延々と続く。AIのレイアウト公式はこれ1つ。

| AIがやること | 代わりにやること |
|---|---|
| 3カラムカードの繰り返し | 1カラム、2カラム、リストを混ぜる |
| 毎セクション「見出し→説明→カード3枚」 | セクションごとに構造を変える |
| 全要素センター揃え | 基本は左揃え。センターはヒーローだけ |
| padding/marginが全部同じ | 重要度で余白に強弱をつける |

### 4. コピー・マイクロコピー

AIのコピーは曖昧で、バズワードだらけで、具体的なことを何も言わない。

| AIが書くこと | 代わりに書くこと |
|---|---|
| 「〜を、もっと〜に」 | 何ができるかを具体的に |
| 「シームレスな体験」 | 体験の中身を書く |
| 「詳しくはこちら」 | 「費用を計算する」「3つを比較する」 |
| 「機能」「ソリューション」 | 「追跡できること」「週3時間節約する方法」 |
| 3文のサブタイトル | 1文以内。長いなら本文に |

### 5. 画像・アイコン

AIは絵文字をアイコン代わりにして、キーワードだけで画像を選ぶ。意味は見ない。

| AIがやること | 代わりにやること |
|---|---|
| 絵文字アイコン（🚀💡✨） | SVGアイコン（Lucide, Heroicons）か、アイコンなし |
| 内容と合わないストック写真 | 公式サイトの画像。なければ画像なしレイアウト |
| 同じ画像の使い回し | ページごとに適切な画像を用意 |
| AI生成イラスト（指6本、文字溶け） | 使わない。SVGイラストか実写 |
| グラデーションプレースホルダー | 画像なしでデザインする |

### 6. UX・インタラクション

AIはページに全部詰め込んで、全要素にアニメーションをつける。

| AIがやること | 代わりにやること |
|---|---|
| 50件以上を一気に表示 | 6-9件表示＋「もっと見る」 |
| 全要素にスクロールアニメ | 使わないか、十分にテスト |
| 全コンポーネントが同じ見た目 | セクションごとにデザインを変える |
| モバイルのタップ領域が小さい | 最低44×44px |
| CTAボタン3つ以上 | メイン＋サブの2つまで |

## 5秒テスト

30項目は大変？じゃあ3つだけ試してほしい。

1. **サイトのスクリーンショットを撮る。** 他のSaaSランディングページと見分けがつくか？
2. **テキストを全部消す。** レイアウトだけで個性があるか？それともただのカードの並び？
3. **知り合いに見せる。** 「これテンプレじゃない？」と聞く。

どれかに「はい」なら、改善の余地がある。

## よくある質問

### GoogleはAI生成サイトにペナルティを与えるの？

GoogleのHelpful Content Updateは、作り方ではなくコンテンツの質を見ている。ちゃんと価値のあるAI生成サイトはペナルティを受けない。ただし、テンプレそのままの見た目は「手抜き」のシグナルとして順位に影響する可能性がある。

### AI生成サイトを人間っぽくする最速の方法は？

フォントを変える、グラデーションを消す、レイアウトにバリエーションを持たせる。この3つだけで最も目立つAIパターンは消える。

### AI生成のイラストは使っていい？

基本的にNG。現在のAI画像生成は指の数がおかしかったり、テキストが溶けたり、ライティングが不自然だったりする。ユーザーはすぐに気づく。SVGイラストか実写がベター。

### AIツールを使わないべき？

いや、AIはスピードを出すのに最高。大事なのは、AIの出力を「最終版」ではなく「初稿」として扱うこと。こういうチェックリストを通して、意図的にデザインを選んで、自分のサイトに見えるまでカスタマイズする。
