Claude Code is Anthropic's terminal-based AI coding agent, and the pricing confuses almost everyone because it draws from your regular Claude subscription. This guide breaks down exactly what you get on Claude Code free vs paid in 2026 — the Pro, Max 5x, and Max 20x tiers, the 5-hour usage windows, and when paying $100 or $200 actually makes sense.

## Free vs Paid at a Glance

| Plan | Price | Usage | Best for |
|------|-------|-------|----------|
| Free tier | $0 | Minimal | Trying it out |
| Pro | $20/mo | Light coding | Occasional sessions |
| Max 5x | $100/mo | ~88k tokens / 5h | Daily developers |
| Max 20x | $200/mo | ~220k tokens / 5h | Power users |
| API | Pay-per-token | Unlimited | Automation / teams |

## How Claude Code Billing Works

The key thing to understand: Claude Code **shares the same token budget as your Claude plan**. There's no separate Claude Code subscription — your Pro or Max plan determines how much you can do. Usage is metered in rolling **5-hour windows**, so limits reset throughout the day rather than monthly.

If you just want to test it, the free tier and Pro get you started, but serious coding sessions exhaust them quickly.

## What Pro ($20/month) Gives You

Pro is the entry point for light, occasional Claude Code use:

- Access to Claude Code from the same $20 Claude Pro subscription
- Enough headroom for short sessions and small tasks
- Shared budget with your regular Claude chat usage

The limitation is depth — a single long agentic coding session can consume your window. Pro is for dabbling, not all-day building.

## What You Unlock With Max ($100 / $200)

Max is where Claude Code becomes a daily driver:

- **Max 5x ($100/mo):** roughly **88,000 tokens per 5-hour window** — for developers who regularly hit Pro's limits
- **Max 20x ($200/mo):** roughly **220,000 tokens per 5-hour window** — the top individual tier

A major advantage: in heavy Claude Code sessions, **over 90% of tokens are cache reads**, which would cost $0.50/M on the API but are included in the flat Max rate. Power users report Max can save up to **93% versus raw API costs**. Full tiers in our [Claude Code pricing guide](/blog/claude-code-pricing-2026/).

Because limits are per 5-hour window, your real constraint is session intensity, not a monthly cap. Compare this subscription model against per-ACU billing in our [Devin vs Claude Code breakdown](/blog/devin-vs-claude-code-2026/).

## Which Should You Pick?

- **Curious and occasional?** Free tier or Pro at $20.
- **Coding daily and hitting Pro's ceiling?** Max 5x at $100.
- **Running long agentic sessions all day?** Max 20x at $200 — still far cheaper than equivalent API usage.
- **Automating or building for a team?** Use the API and pay per token.

## FAQ

**Is Claude Code free?** There's a free tier, but it's minimal — meaningful coding requires at least Pro ($20).

**Does Claude Code cost extra on top of Claude?** No — it uses the same token budget as your Pro or Max subscription.

**What are the 5-hour windows?** Usage limits reset on a rolling 5-hour basis rather than monthly, so you regain capacity through the day.

**Is Max worth $100–200 over the API?** For interactive coding, yes — cache-heavy sessions make Max up to ~93% cheaper than equivalent API spend.
