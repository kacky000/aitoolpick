# Upstash Review 2026: Is It Worth It?

Upstash is serverless data platform providing redis-compatible caching, kafka-compatible messaging, and vector database with per-request pricing and global replication. In this review, we cover everything you need to know before subscribing.

## What is Upstash?

Upstash is serverless data platform providing redis-compatible caching, kafka-compatible messaging, and vector database with per-request pricing and global replication. It is designed for serverless developers, next js developers, edge functions and anyone looking for a reliable solution in the database space.
